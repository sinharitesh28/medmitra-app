# Security Policy

## Supported Versions

The following versions of MedMitra are currently supported with security updates:

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability

We take the security of MedMitra seriously. If you have discovered a security vulnerability, please do not disclose it publicly until it has been addressed.

### How to Report

Please report security vulnerabilities by emailing **rityasinha@gmail.com**.

In your report, please include:
- A description of the vulnerability.
- Steps to reproduce the issue.
- Potential impact.

### Response Timeline

- You can expect an acknowledgment of your report within **48 hours**.
- We will provide a timeline for the fix after the initial assessment.
- Once the vulnerability is resolved, we will release a patch and notify you.

Thank you for helping keep MedMitra secure!
